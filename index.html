1ST PROJECT · TEAM LEAD

가상 금융회사 네트워크 및 웹사이트 구축

2026

가상의 금융회사를 주제로 Rocky Linux 기반의 서버와 네트워크 환경을 구성하고, 회사 웹사이트와 데이터베이스를 구축한 프로젝트입니다.

  • 팀장으로서 프로젝트 일정과 역할을 조율했습니다.
  • Rocky Linux 기반 웹 서버와 데이터베이스 서버를 구성했습니다.
  • 가상 금융회사 환경에 맞춰 네트워크 구조와 서비스 구성을 설계했습니다.
  • 각 서버와 웹사이트, 데이터베이스의 연결 상태를 점검했습니다.
Rocky Linux Network Web Server Database Team Lead

프로젝트 목적

가상회사의 네트워크와 서버 환경을 직접 설계하면서 웹 서비스와 데이터베이스가 연결되는 전체 구조를 이해하는 것이 목적이었습니다.

담당 역할

팀장으로서 역할과 일정을 조율하고, 팀원별 결과물이 하나의 회사 환경에서 정상적으로 연결되는지 전체 진행 상황을 확인했습니다.

2ND PROJECT · TEAM MEMBER

2026

  • 서버·네트워크·보안 구성 요소가 연결되는 구조를 학습했습니다.
  • 오류 발생 시 설정과 통신 상태를 순서대로 확인했습니다.

제작 경험

단순히 어렵게 만드는 데 집중하기보다, 참가자가 단서를 확인하며 의도한 보안 개념에 도달할 수 있도록 문제의 조건과 풀이 순서를 구성했습니다.

풀이 성과

  • 다른 팀이 제작한 총 100문제 중 38문제를 해결했습니다.

담당 역할

팀원별로 제작한 기능과 취약점을 하나의 교육 포털 서비스로 통합하고, 기능 충돌과 데이터베이스 연결, 테스트 결과와 제출 자료를 전체적으로 점검했습니다.

성과

OWASP Juice Shop 취약점 분석 과제

2026

부트캠프 웹 보안 과제로 OWASP Juice Shop 환경의 취약점을 분석하고 문제를 해결했습니다.

  • 요청·응답과 데이터 처리 흐름을 확인해 취약점의 발생 원인을 분석했습니다.
  • 주요 취약점은 재현 과정, 영향과 대응 방안을 보고서로 작성했습니다.

실습 방식

입력값과 요청·응답, 권한 검증, 데이터 처리 흐름을 확인하며 취약점이 발생하는 지점을 단계적으로 좁혀갔습니다.
